Cartella per le texture PNG usate da [tab.texture].

Metti qui i tuoi file PNG e riferiscili in theme.conf:

    [tab.texture]
    file = "textures/nome_file.png"

Suggerimenti:
- Per il tiling usa immagini ≤ 64×64 px
- PNG in scala di grigi funzionano bene con blend_mode=multiply o overlay
- Usa opacity tra 0.2 e 0.5 per risultati naturali
